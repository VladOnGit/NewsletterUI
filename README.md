# [NewsletterUI.github.io](https://vladongit.github.io/NewsletterUI/)

O interfață de utilizator (UI) elegantă și responsivă pentru un formular de abonare la newsletter. Proiectul este construit cu HTML și CSS, concentrându-se pe un design modern și o experiență vizuală plăcută.

##  Caracteristici
* Design Modern, Temă dark elegantă (Dark Mode) cu accente de culoare.
* Animații CSS: Fundal cu gradient animat care se schimbă fluid.
* UI/UX: Efect de blur pe container (Glassmorphism).
* Iconițe integrate: Se utilizează FontAwesome pentru o reprezentare vizuală clară a informațiilor.
* CSS Modern: Cod organizat folosind variabile CSS (`:root`) pentru o modificare rapidă a paletei de culori.

##  Tehnologii Folosite
* HTML, CSS.
